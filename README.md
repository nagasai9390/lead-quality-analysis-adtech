# Lead Quality Analysis — Ad Tech Case Study

## Overview
Analysis of 3,021 leads from a programmatic advertising dataset 
to identify lead quality drivers and revenue optimization opportunities.

## Tools Used
- Excel — Pivot tables and charts
- Python — Pandas, Matplotlib, Jupyter Notebook
- SQL — SQLite with subqueries

## Key Findings
- Identified AdKnowledge as the highest quality traffic source (12% vs 8% baseline)
- 300x250 widget format closes at 16% vs 6-7% for 302x252 format
- Mid-range debt ($10K-$90K) is the sweet spot for lead quality
- Quantified $9,063 revenue uplift opportunity through smarter traffic allocation

## Structure
- `lead_quality_analysis.ipynb` — Full Python and SQL analysis
- `leads.csv` — Raw dataset (3,021 leads)
- `aarki_lead_analysis.docx` — Full written report

## Results Validation
All findings were independently validated across Excel, Python, and SQL 
— identical results across all three platforms.
